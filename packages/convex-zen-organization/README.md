# convex-zen-organization

Organization plugin for `convex-zen`.

## Install

```bash
npm install convex convex-zen convex-zen-organization
```

## Usage

```ts
import { defineConvexZen } from "convex-zen";
import { organizationPlugin } from "convex-zen-organization";

export default defineConvexZen({
  plugins: [organizationPlugin()],
});
```
